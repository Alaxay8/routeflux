# TUI Flow

## Main Screen
- Shows connected, disconnected, or auto state
- Highlights the active subscription and active node
- Lists all imported subscriptions with counts and parser state
- Lists nodes for the selected subscription with protocol and latency

## Actions
- `c`: manual connect
- `a`: automatic selection
- `r`: refresh subscription
- `d`: disconnect
- `s`: open settings

## Settings Screen
- Displays current refresh and probe settings
- Lets the user cycle refresh interval presets with `+` and `-`
- Returns to the main screen with `s`
